# CLI Expense Tracker

A simple, interactive command-line application built with Python to help manage personal finances. This project was built as part of a roadmap.sh backend challenge.

##  Features
- **Add Expenses:** Quickly log a description and the amount spent.
- **View History:** See a formatted list of all your recorded expenses.
- **Delete Entries:** Remove specific expenses using their unique ID.
- **Summary:** Get a quick calculation of your total spending.
- **Persistent Storage:** All data is saved to a `expenses.json` file so it's there when you come back.
